Firebase設定（データ共有）
Firebase は完全無料で使えます Googleが提供するデータベースサービス。家族での利用レベルでは料金は一切発生しません（無料枠：1GB・月10GBまで）。
設定手順
  1. 1
    Googleアカウントでサインイン
    console.firebase.google.com にアクセスし、Googleアカウントでログイン
  2. 2
    プロジェクトを作成
    「プロジェクトを追加」→ 名前を入力（例：投資管理）→ Googleアナリティクスはオフでも可 → 「プロジェクトを作成」
  3. 3
    ウェブアプリを追加
    プロジェクト画面の「</>」ボタン → ニックネームを入力 → 「アプリを登録」→ 表示された firebaseConfig の { } 全体 をコピー
  4. 4
    Realtime Database を作成
    左メニュー「構築」→「Realtime Database」→「データベースを作成」→ ロケーションは「asia-southeast1」→「テストモードで開始」→「有効にする」
  5. 5
    APIキーだけ下のフォームに入力して保存
    Firebase Console →「設定（⚙）」→「プロジェクトの設定」→「全般」→「マイアプリ」→ apiKey の値をコピーして下に貼り付け
以下の項目はすでに入力済みです。apiKey だけ入力してください。
マイルストーンを追加
前のマイルストーンに上乗せする金額を入力してください。
例）1つ目¥1,500万・2つ目¥500万 → 2つ目までに合計¥2,000万必要
